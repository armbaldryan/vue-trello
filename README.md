# vue-trello
Testing Vue.js 

Demo

https://vue-trello-arm.firebaseapp.com/

You must include fbConfig.js in src folder and write your firebase configs inside that file like

```sh
export const fbConfig = {
  ...
};
```

## Quick Overview

```sh
npm i
npm run serve
```
